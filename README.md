# ITS-PB Lab Suite

Единая исходная кодовая база для комплекса генераторов ИЛ «ИТС-ПБ».

## Модули

- ICP OS — генерация отчетов ICP-OES.
- AAS — генерация отчетов ААС.
- Токсичность — генерация протоколов токсичности.
- Хроматография — зарезервировано для следующего модуля.

## Запуск из исходников

1. Установить зависимости:

```bat
python -m pip install -r requirements.txt
```

2. Запустить приложение:

```bat
python app\main.pyw
```

## Сборка EXE

```bat
BUILD_EXE_ONCE.bat
```

После сборки готовая папка будет в `dist\ITS-PB Lab Suite`.
Сотрудникам передается вся папка `dist\ITS-PB Lab Suite`.

## Важный принцип

Теперь все изменения вносятся в эту единую кодовую базу, а не собираются «с нуля» отдельными архивами.
