AAS RELEASE 0.3.2

Included:
- Al, Ag, Zn migrated from the newly supplied reports;
- Cu, Ni, Co migrated from the previously supplied reports;
- all six use the shared RTF clone engine;
- 2–5 dynamic parallel rows;
- unique row_N.action_time for every measurement;
- updated AAS Excel template with columns Al, Cu, Ni, Ag, Zn, Co, Hg.

Installation:
1. Close the application.
2. Put AAS_RELEASE_0_3_2 inside ITS-PB-Lab-Suite.
3. Run INSTALL.cmd.
4. Run RUN_FROM_SOURCE.bat and test.
5. Commit the updated project.
6. Run PREPARE_RELEASE.cmd only after the working tree is clean.
