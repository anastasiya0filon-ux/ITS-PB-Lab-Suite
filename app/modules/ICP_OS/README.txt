ICP OS generator — ИЛ ИТС-ПБ

Запуск:
1. Распакуйте архив.
2. Запустите START.bat.
3. При необходимости нажмите «Создать шаблон Excel».
4. Заполните Excel: первый столбец — «Шифр образца», далее элементы Pb, Cr, Ni, Co, Cu, As, Sn, Se, Sb, Ba, Al, Fe, Zn, Mn, Cd, Ag.
5. Выберите Excel, оператора, дату и время первого анализа.
6. Нажмите «СФОРМИРОВАТЬ ОТЧЕТЫ».

Для каждой строки Excel будет создан отдельный DOCX только по заполненным элементам.
Позиции виал генерируются 1–105, первая и вторая позиции в отчете отличаются.
Время второго блока: +307–350 секунд.
Следующий документ: +180–450 секунд после второго блока предыдущего документа.
