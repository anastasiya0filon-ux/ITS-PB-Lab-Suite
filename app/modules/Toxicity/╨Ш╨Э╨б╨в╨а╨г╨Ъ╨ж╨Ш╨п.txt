ГЕНЕРАТОР ПРОТОКОЛА ТОКСИЧНОСТИ

Что изменено в этой версии:
1. Контрольная линия больше не является постоянным шаблоном.
   Для каждого протокола заново генерируется контрольная серия по статистической модели реальных протоколов.

2. Значения Кн/Оп генерируются нестационарно:
   - каждая строка убывает по циклам;
   - 8-й цикл всегда 0;
   - опыт строится от конкретного контроля, а не от фиксированной кривой.

3. S считается из таблицы:
   S = round(0.75 * сумма значений по 8 циклам).

4. Is считается автоматически:
   Is = S_опыт_ср / S_контроль_ср * 100.

5. График строится из средних значений таблицы:
   - контроль = среднее по 5 столбцам Кн для каждого цикла;
   - опыт = среднее по 4 столбцам Оп для каждого цикла.

КАК ПОЛЬЗОВАТЬСЯ

Вариант для Python:
1. Запустите START_ГЕНЕРАТОР_ДЛЯ_PYTHON.bat
2. Введите номер, дату, шифр образца, быка, It.
3. Нажмите "Сформировать Word-протокол".

Сборка EXE:
1. На компьютере, где установлен Python, запустите СОБРАТЬ_EXE_ОДИН_РАЗ.bat
2. Готовый файл появится в папке dist: Токсичность.exe
3. Этот EXE можно отправлять сотрудникам. Python им не нужен.

Файлы на выходе:
- Word-протокол DOCX;
- PNG-график;
- JSON с расчетами и проверками;
- CSV проверки соответствия графика таблице.
