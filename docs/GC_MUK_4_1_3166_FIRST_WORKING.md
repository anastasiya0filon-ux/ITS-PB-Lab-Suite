# GC МУК 4.1.3166 — первый рабочий блок

Реализовано:

- четыре режима интерфейса: одиночный/массовый × рандом/фактические;
- постоянный перечень из 28 компонентов;
- количество хроматограмм жестко определяется методикой и равно 2;
- отдельный расчет ПИД-1 и ПИД-2;
- расчет площади, высоты, sigma и времени удерживания;
- жесткое окно времени удерживания ±5%;
- интервал 2940–3540 с между двумя хроматограммами одного образца;
- интервал 6480–6600 с между последовательными образцами серии;
- PNG полного графика, JSON и CSV;
- подготовлена архитектура для следующего подключения печатной формы.

Текущий результат на один образец: 4 PNG (2 хроматограммы × 2 детектора).
