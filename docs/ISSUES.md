# Реестр замечаний ITS-PB Lab Suite

## Закрыто в v0.3

### DOC-002 — формат числовых значений ICP OS
Для концентраций и итогов используется единое правило печати:
- `0.031 → 0.0310`
- `0.21 → 0.2100`
- `0.00362 → 0.0036`
- `1.23665 → 1.237`
- `26791 → 2679`
- `26795 → 2680`
- `267918 → 2679`

### GEN-003 — явные скобки в нелинейных формулах
Формулы интенсивности для элементов с коэффициентом `c` приведены к виду:
`round((a + (b * conc_1)) / (1 + (c * conc_1)))`.

### ICP-004 — массовый режим «Фактические значения»
Добавлена третья вкладка ICP OS. Пользователь загружает Excel со значениями `conc_2` и `conc_4`; программа рассчитывает `conc_1`, `conc_3`, интенсивности, СКО, время и позиции.

## В работе / проверка

### DOC-001 — оформление ICP OS
Максимально приблизить шрифты, табуляции, поля, интервалы и положение элементов к оригиналу прибора.

### UI-001 — сборка в EXE
Подготовить распространение как Windows-приложение без установки Python у сотрудников.

## Закрыто в v0.3.1-draft

### AAS-METHOD-001 — профили методик ААС
Добавлен выбор методики в интерфейсе ААС. Методика определяет доступные элементы и параметры выгружаемых первичек прибора: строку `Метод`, файл концентрации, коэффициент расчета и при необходимости объем пробы.

### AAS-BATCH-001 — методика для массовой генерации
В массовой генерации ААС методика выбирается один раз в интерфейсе, а Excel содержит только данные серии по этой методике. Это снижает риск смешения ГОСТ 31870, ГОСТ 31950 и ПНД Ф 14.1:2:4.140-98 в одной серии.
