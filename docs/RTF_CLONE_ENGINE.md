# Универсальный RTF Clone Engine

## Утвержденный принцип

При наличии оригинального приборного отчета документ не создается заново.
Программа клонирует эталонный RTF и меняет только переменные.

## Движок сохраняет

- поля страницы;
- шрифты и размеры;
- табуляцию;
- интервалы;
- границы;
- внутренние поля ячеек;
- встроенные изображения;
- структуру приборного документа.

## Движок умеет

- заменять одно или все вхождения эталонного значения;
- заменять значение строки после двоеточия;
- переносить длинный текст внутри существующей ячейки;
- формировать 2-5 строк результатов;
- вписывать широкие таблицы в печатную область через `cellx`;
- сохранять результат в исходном формате RTF.

## Первый потребитель

`app/modules/AAS/aas_rtf_clone.py`

После проверки на AAS движок можно подключать к другим приборным модулям.
