# AAS release 0.3.3

## Fixed
- Individual timestamps for every parallel measurement remain enabled.
- Interval between consecutive parallels is fixed in the range 9:00–10:00.
- The report field `Метод` is now an instrument property of the element template/config.
- It is not derived from ГОСТ, ПНД Ф, or another normative method profile.

## Instrument method values
- С модификатором: Cd, Pb, Sb.
- Стандартный: Ag, Al, As, Ba, Co, Cr, Cu, Fe, Hg, Mn, Ni, Se, Sn, Ti, Zn.

## Elements
Ag, Al, As, Ba, Cd, Co, Cr, Cu, Fe, Hg, Mn, Ni, Pb, Sb, Se, Sn, Ti, Zn.
