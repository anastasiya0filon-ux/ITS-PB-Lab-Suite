# Выбор методики в GC-модуле

Выпадающий список **Методика** является единственной точкой выбора НД.

От выбранного НД зависят:

1. математический профиль: вещества, времена удерживания, калибровочные зависимости,
   площади, высоты и валидаторы;
2. профиль отчёта — только если конкретный НД действительно отличается по структуре.

От выбранного НД не зависят общие свойства Хроматэк-Кристалл 5000:
шрифты, цвета, оси, базовая геометрия, стили линий и приборное оформление.

Незавершённые методы не показываются пользователю. Для подключения нового НД нужно:

- создать его математическую конфигурацию/адаптер;
- при необходимости создать отдельный report_profile;
- установить в methods/registry.json поля enabled=true и implemented=true.
