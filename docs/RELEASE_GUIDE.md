# Как выпустить релиз ITS-PB Lab Suite

## 1. Проверить состояние проекта
```cmd
git status
```
Должно быть: `nothing to commit, working tree clean`.

## 2. Собрать приложение
Запускать сборку только на Python 3.11:
```cmd
python --version
```
Должно быть: `Python 3.11.9`.

Затем запустить сборочный BAT текущей версии проекта.

## 3. Создать тег версии
Например:
```cmd
git tag v0.3.1
git push origin v0.3.1
```

## 4. Создать GitHub Release
На странице репозитория открыть:
`Releases` → `Draft a new release`.

Заполнить:
- Tag: `v0.3.1`
- Title: `ITS-PB Lab Suite 0.3.1`
- Description: кратко перечислить изменения из `CHANGELOG.md`.

Во вложения добавить готовый архив или установщик для сотрудников.

## 5. Что отправлять сотрудникам
Сотрудникам отправлять только готовый релиз: portable ZIP или Setup.exe. Исходный код из репозитория сотрудникам не нужен.
