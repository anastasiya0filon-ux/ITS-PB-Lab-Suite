# GC MUK 4.1.3166 Core v0.1

Первый программный блок газовой хроматографии.

## Реализовано

- загрузка утвержденного паспорта методики;
- жесткое количество хроматограмм: 2;
- интервалы времени:
  - 2940–3540 с между хроматограммами одного образца;
  - 6480–6600 с между образцами серии;
- загрузка отдельных моделей ПИД-1 и ПИД-2;
- расчет площади и высоты;
- расчет sigma;
- положение максимума в жестком окне ±5% от среднего tR;
- JSON-паспорт каждого пика;
- уровни неидеальности 0–3;
- построение четырех PNG на образец:
  - хроматограмма 1 / ПИД-1;
  - хроматограмма 1 / ПИД-2;
  - хроматограмма 2 / ПИД-1;
  - хроматограмма 2 / ПИД-2;
- режимы random и actual;
- два отдельных Excel-шаблона.

## Пока не подключено

- экран PySide6 в `app/main.pyw`;
- чтение массовых Excel-шаблонов;
- точная приборная стилизация PNG по эталонному отчету;
- вставка PNG в печатную форму через RTF Clone Engine.

## Проверка

Из корня репозитория:

```cmd
python TEST_GC_CORE_V0_1.py
```

Результат появится в:

```text
app/modules/GC/output/test_v0_1
```

## Важное замечание

Алгоритм режима `random` пока использует симметричное отклонение 1–5% от
базовой концентрации. Это изолировано в одной функции `_random_pair` и может
быть заменено без изменения моделей, рендерера и интерфейса.
