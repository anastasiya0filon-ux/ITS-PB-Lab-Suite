# GC Passport 1.0

Установлен независимый паспорт типографики Chromatek.

## Замороженные части

- signal.py
- peaks.py
- scale_profiles.py
- compositor.py
- расчёт RT, площадей, высот и концентраций
- baseline и шум
- формирование DOCX

## Управляемые паспортом параметры

- размер шрифта осей
- размер аналитических подписей
- шаблон подписи
- зазор от вершины пика
- окно поиска физического максимума
- политика пересечений

Параметры находятся в `app/modules/GC/chromatek_renderer/typography_passport.json`.
