# AAS: правила выбора методики

## Главное правило

- **Элемент** определяет шаблон Word, расчетную модель, график, строку `Метод`, коэффициент расчета и параметры прибора.
- **Методика** ограничивает список доступных элементов и формирует строку `файл концентрации`.
- Строка `Метод` в шаблоне не меняется при выборе нормативной методики.

## Формирование строки `файл концентрации`

Формат:

```text
<префикс методики><символ элемента>.cnc
```

Примеры:

```text
ГОСТ 31870-2012 + Cd -> ГОСТ 31870Cd.cnc
ПНД Ф 14.1:2:4.140-98 + Cd -> ПНД Ф 140-98Cd.cnc
ГОСТ 31950-2012 + Hg -> ГОСТ 31950Hg.cnc
```

## Доступность элементов

В интерфейсе показываются только элементы, для которых в проекте уже есть:

1. шаблон `templates/<Element>.docx`;
2. расчетная конфигурация `configs/<Element>.json`.

Полный справочник методик может содержать больше элементов; они станут доступны после добавления шаблона и JSON-конфигурации.
