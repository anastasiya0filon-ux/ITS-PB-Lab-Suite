# AAS RTF Clone Engine

Отчет формируется из оригинального RTF прибора. Движок не строит документ заново: сохраняются шрифты, интервалы, таблицы, границы, поля страницы и встроенная графика. Меняются только переменные, расчетные значения и число строк измерений (2–5).

## Установка

1. Распаковать архив поверх корня репозитория.
2. Выполнить: `python INSTALL_AAS_RTF_CLONE.py`
3. Запустить: `python app\main.pyw`

Перед изменением установщик сохраняет резервные копии `main.pyw.before_rtf_clone` и `aas_report_generator.py.before_rtf_clone`.
