# Chromatek Profiles 1.0

## Центральное правило

Выбор методики в интерфейсе определяет:

1. математический профиль;
2. профиль отчёта, только если конкретный НД требует отличий.

Выбор методики не изменяет:

- шрифты и типографику Хроматэка;
- цвета ПИД-1 и ПИД-2;
- геометрию хроматограммы;
- базовый стиль baseline;
- общий алгоритм рендера;
- оформление прибора и прошивки.

## Подключение нового НД

Для нового метода добавляются:

- запись в `methods/registry.json`;
- математический модуль/конфигурация;
- при необходимости отдельный JSON в `profiles/reports/`.

Если отчёт не отличается, используется `report_profile: chromatek_standard`.
