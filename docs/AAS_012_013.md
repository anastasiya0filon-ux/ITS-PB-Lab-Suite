# AAS-012 / AAS-013

## AAS-012 — время анализов МГА

- В шапке выводится время более позднего анализа без секунд.
- В таблице строка 1 — более поздний анализ.
- Каждое следующее измерение выполняется раньше предыдущего на 9–10 минут.
- Интервал воспроизводимый и чаще близок к 9 минутам 30 секундам.
- При массовой генерации время следующего отчета сдвигается вперед на 9–10 минут.

## AAS-013 — параллельные измерения

- В интерфейс добавлено поле «Параллельные измерения».
- Значение по умолчанию: 2.
- Допустимые значения: 2–5.
- Таблица Word динамически получает нужное количество строк.
- Среднее, СКО абсолютное и СКО относительное рассчитываются по фактическому числу измерений.

## Установка

Распаковать архив в корень репозитория ITS-PB-Lab-Suite с заменой файла.
После проверки выполнить:

```cmd
git status
git add app/modules/AAS/aas_report_generator.py docs/AAS_012_013.md
git commit -m "AAS: add measurement count and analysis timing"
git push
```
