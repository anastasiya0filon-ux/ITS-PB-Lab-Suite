# Chromatek Multi-Method Platform 1.0

## Цель

Один общий эмулятор Хроматэк-Кристалл 5000 для МУК 4.1.3166, 4.1.3169,
4.1.3170, 4.1.3171 и последующих НД.

## Разделение ответственности

### Метод НД
Хранит вещества, RT, калибровочные зависимости, правила расчёта площади/высоты,
валидацию входных данных. Не рисует, не выбирает шрифты и не формирует DOCX.

### Chromatek Platform
Хранит универсальный контракт данных, реестр методов и выбор профиля прибора.

### Chromatek Renderer / Passports
Общие для всех НД: геометрия, оси, цвета ПИД-1/ПИД-2, типографика,
baseline-стиль, правила подписей и внешний вид хроматограммы.

### Report Engine
Общий макет отчёта Хроматэк. Получает унифицированный результат метода.

## Добавление нового НД

Создаётся только папка:

`app/modules/GC/methods/MUK_4_1_3170/`

с descriptor.py, adapter.py, calibration_model.json и validator.py. Общий рендер
не копируется и не изменяется.

## Текущий этап

3166 подключён как legacy-adapter. Его рабочие файлы не перемещены, чтобы не
нарушить release/0.4.0-rc1. Следующая миграция должна вынести расчёт 3166 за
универсальный контракт с контрольным сравнением результатов до/после.
