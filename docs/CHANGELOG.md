# Changelog

## v0.3
- ICP OS: добавлен режим «Фактические значения» для массовой генерации через Excel.
- ICP OS: добавлен шаблон Excel с парами `conc_2` / `conc_4` для каждого элемента.
- ICP OS: добавлена функция `generate_from_actual_excel`.
- ICP OS: добавлена функция `make_item_from_actual`.
- DOC: исправлен формат вывода концентраций и итогов по правилу DOC-002.
- GEN: нелинейные формулы интенсивности записаны с явными скобками.
- QA: выполнена тестовая генерация DOCX и рендер страницы.

## v0.2
- Стартовая рабочая база Source Project.
- Исправлена строка `Оператор:` в ICP OS.
- Добавлен базовый Qt-интерфейс.

## v0.3.1-draft
- AAS: добавлен справочник профилей методик: ГОСТ 31870, ГОСТ 31950, ПНД Ф 14.1:2:4.140-98.
- AAS: в блок общих параметров добавлен выпадающий список «Методика».
- AAS: список элементов теперь зависит от выбранной методики.
- AAS: массовая генерация через Excel использует одну выбранную методику для всей серии.
- AAS: методика управляет текстом строки «Метод», файлом концентрации `.cnc`, коэффициентом расчета и объемом пробы при наличии переопределений.
